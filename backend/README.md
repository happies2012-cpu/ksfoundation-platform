# KSF AI Backend

This is the Python-based backend for the KSF Foundation platform.

## Features
- **FastAPI** high-performance API
- **AI Agents** with LangGraph/PydanticAI
- **Data Layer** (Google, Social, Commerce)
- **Cloud Automation** (SSH, K3s, Docker)

## Running
```bash
uv run fastapi dev app/main.py
```
